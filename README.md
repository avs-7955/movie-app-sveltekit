## Run Locally

Clone the project

```bash
  git clone https://github.com/avs-7955/movie-app-sveltekit.git
```

Go to the project directory

```bash
  cd movie-app-sveltekit
```

Install the packages

```bash
  npm i
```

Run the server

```bash
  npm run dev
```

Open the localhost shown in the console.
